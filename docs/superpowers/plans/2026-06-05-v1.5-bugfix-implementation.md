# v1.5 Bugfix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix the v1.5 regressions in high-DPI window layout, daily rest count semantics, and manual pause behavior after system sleep/wake or lock/unlock.

**Architecture:** Keep the fixes narrowly scoped. Persist daily stats through a small JSON store owned by `DailyRestStats`, keep pause semantics in `TrayAppContext` by preserving `Paused` state during system rest events, and make the About/reminder windows use resilient WinForms layout without changing the reminder window shape or broader app theme.

**Tech Stack:** .NET 10, WinForms, xUnit, JSON file persistence.

public bool MarkRestSessionCompleted(Guid sessionId)
{

    if (!_completedSessionIds.Add(sessionId))
